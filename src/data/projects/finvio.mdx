## Overview

Finvio is an AI Powered Invoicing and Analytics System (Ongoing / Development Phase).

- Architected a scalable backend system using Next.js 15 Server Actions, TypeScript, PostgreSQL, and Prisma ORM, designing normalized relational schemas (10+ tables) to support high-volume invoice storage, financial aggregations, and real-time analytics queries with <200ms response times.
- Implemented an AI-driven invoice intelligence pipeline by integrating Google Gemini APIs to perform context-aware field extraction, invoice autofill, error detection, and recommendation generation, improving data accuracy by 25–30% and reducing manual input.
- Built a secure document generation and storage service leveraging Puppeteer for deterministic PDF rendering and Base64 asset encoding for logos/signatures, combined with Clerk-based authentication, RBAC, and multi-step onboarding, enabling safe access to sensitive financial data.
