## Overview

Inscribe is an AI Powered README Generator.

- Delivered an AI README generator that streams responses via a Next.js serverless endpoint (AI SDK + Web Streams), reducing manual docs by 95%.
- Implemented secure OAuth with NextAuth and integrated GitHub/GitLab REST APIs to ingest repo metadata, files, and contributors, normalizing cross-platform data for consistent processing.
- Made the app production-ready with Postgres user persistence, CORS/security-headers middleware, and JSON-LD/analytics; the parser lifted documentation completeness to 90%.
