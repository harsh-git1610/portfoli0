## Overview

Rewear is a Sustainable Fashion Exchange Platform.

- Engineered a sustainable exchange platform with dual trading systems (direct swaps + points), extending garment lifecycles for 50+ active users and reducing textile waste.
- Develped a real-time dashboard for profiles and a points system using React and Web APIs, instrumented events, and shipped gamified incentives, boosting user engagement by 40% across active traders.
- Designed an optimized listing flow and marketplace UX, reducing listing time by 60% and increasing trade completion rates.
